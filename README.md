# vue-rollup-example

A Vue + Rollup Boilerplate and a big help from `rxjs` and `ora` to make a pretty CLI output.

```shell
git clone https://github.com/jonataswalker/vue-rollup-example.git
cd vue-rollup-example
npm install
npm run dev
```
